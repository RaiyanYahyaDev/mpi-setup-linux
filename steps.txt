Setting up Master Image:

	docker pull ubuntu
	docker run -t -d ubuntu

create another container with the same command

use this to login:
	docker exec -it <container name> /bin/bash

rename the contianer to master and slave:

	docker container rename a287a95975b2 slave

create ssh login between both the containers :
	https://support.dbvisit.com/hc/en-us/articles/360000370515-Setup-Passwordless-SSH-between-two-servers-

be sure to install ssh servers as well:
	apt install openssh-server
	service ssh restart
